# vibescan

Security Scanner for Vibe-Coded Software — SAST + DAST + exploit simulation + attack path analysis.

## Install

```bash
pip install vibescan
# or
pipx install vibescan
```

## Usage

```bash
vibescan run .          # scan current directory
vibescan scan <path>    # scan a specific path
vibescan doctor         # check tool availability
```
